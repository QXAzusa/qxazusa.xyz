# 校對
## 能力要求

具有較強的外語理解能力

能夠熟練使用字幕製作軟體

語言錶達能力較好，能夠對不合適的翻譯內容加以潤色

## 工作內容

檢查字幕是否存在時軸錯誤、樣式錯誤、翻譯內容錯誤

通常校對會同時兼任壓製環節的工作

## 工作要求
校對除了遵守翻譯內容和格式規範以外，主要註意以下問題：
1.	漏翻、錯翻、調整長句語序、語句不通、明顯錯別字
2.	時軸錯位、長軸
3.	字體樣式錯誤
4.	修正完成後，務必再完整看一遍，以防漏軸

:::caution

請務必安裝所在字幕組所用的字幕字體，以免出現宋體事故

因為這是視訊內容製作的**最後一步**，所以請務必做好檢查工作！

:::
