# 美術
## 能力要求
具有良好的審美觀

熟練使用Photoshop等軟體


## 工作內容
根據剪輯製作的視訊內容製作封麵，這項工作通常在剪輯開始製作時就應開始進行

根據活動需要製作對應的文宣圖，如：每月艦長福利、生日會、重大活動等

## 工作要求
需要符合發錶平臺對應的審核標準、避免所用素材涉及版權問題
### 封麵要求
|平臺|建議尺寸|最低尺寸|文件大小|
|----|----|----|----|
|嗶哩嗶哩|≥1146x717|960x600|≤5MB|
|YouTube|≥1280x720|-|≤2MB|
### 文宣圖要求
能夠完成準確錶達需要文宣內容的需求，不産生歧義
